Project Overview

Discover Your Academic Future

UC Course Matcher is an interactive platform designed to simplify how students navigate and compare courses across the University of California system. The tool provides two core functions:

1. Intercampus Course Matching which identifies the most semantically similar courses across UC campuses for any selected class.
2. Intracampus Prerequisite Visualization which displays structured prerequisite graphs that clarify course dependencies within a single campus.

By combining text-embedding similarity with dynamic prerequisite visualization, UC Course Matcher unifies fragmented catalog information into an intuitive interface that supports clearer academic planning and exploration.
